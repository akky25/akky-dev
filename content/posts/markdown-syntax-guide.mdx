## リスト

### 順序なしリスト

- 第1レベルのアイテム
- 別の第1レベルのアイテム
  - 第2レベルのアイテム
  - もう一つの第2レベル
    - 第3レベルのアイテム
    - さらに第3レベル
      - 第4レベルのアイテム

* アスタリスクでも可能
* このように使えます
  * ネストも同様

+ プラス記号でも可能
+ 同じように動作します

### 順序付きリスト

1. 最初のアイテム
2. 二番目のアイテム
   1. サブアイテム2.1
   2. サブアイテム2.2
      1. さらにネスト2.2.1
      2. さらにネスト2.2.2
3. 三番目のアイテム

### タスクリスト

- [x] 完了したタスク
- [ ] 未完了のタスク
- [ ] 別の未完了タスク
  - [x] 完了したサブタスク
  - [ ] 未完了のサブタスク

```typescript
// TypeScript のコード
interface User {
  id: number;
  name: string;
  email: string;
  roles: string[];
}

class UserService {
  private users: User[] = [];

  addUser(user: User): void {
    this.users.push(user);
  }

  getUser(id: number): User | undefined {
    return this.users.find(u => u.id === id);
  }
}
```

## 脚注

これは脚注付きのテキストです[^1]。別の脚注もあります[^2]。

[^1]: これは最初の脚注の内容です。
[^2]: これは二番目の脚注の内容です。長い脚注も可能で、
    複数行にわたって記述できます。
